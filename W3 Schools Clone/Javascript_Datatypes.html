Javascript Datatypes

❮ Previous
Next ❯

JavaScript provides different data types to hold different types of values.It can hold any type of values such as numbers, strings etc. For example:

                
Example

                var a = 40; //holding number
                var b = "Rahul"; //holding string

There are two types of data types in JavaScript.

  • Primitive data type
  • Non-primitive data type

    • Javascript has 8 Datatypes

    1. String
    2. Number
    3. Bigint
    4. Boolean
    5. Undefined
    6. Null
    7. Symbol
    8. Object
