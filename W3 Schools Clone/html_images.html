HTML img tag is used to display image on the web page. HTML img tag is an empty tag that contains attributes only, closing tags are not used in HTML image element

                        
Example

                        <img src="good_morning.jpg" alt="Good Morning Friends"/>

    Attributes of HTML img tag

  • src:

    It is a necessary attribute that describes the source or path of the image. It instructs the browser where to look for the image on the server. The location of image may be on the same directory or another server.

  • alt:

    The alt attribute defines an alternate text for the image, if it can't be displayed. The value of the alt attribute describe the image in words.

  • width:

    It specifies the width of the image.

  • height:

    It specifies tyhe height of the image.

img tag as a link

                        
Example

                        <a href=""><img src="robot.jpg" height="100" width="100"><a>
