HTML COMMENTS

❮ Previous
Next ❯

HTML comments are used to add explanatory notes or comments within your HTML code that are not visible in the rendered output of the webpage. Comments can be helpful for providing information to other developers, documenting your code, or temporarily hiding parts of the code for testing or debugging purposes.

HTML comments are enclosed within <!-- and --> markers. Anything between these markers is considered a comment and is ignored by browsers when rendering the page. Here's how you can use HTML comments: 

                        
Example

                        <!DOCTYPE html>
                        <html>
                            <head>  <!--Document Header Starts-->
                                <title>This is document title</title>
                            </head> <!--Document Header Ends -->
                            <body>
                                <p>Document content goes here.....</p>
                            </body>
                        </html>

Multiline Comments

You can comment multiple lines by the special beginning tag <!-- and ending tag --> placed before the first line and end of the last line.

                        
Example

                        <h1>Website Title</h1>
                        <p>This is the introduction to the content.</p>
                        <hr>
                        <h2>Section 1</h2>
                        <!-- <p>This is the content of section 1.</p>
                        <p>This is the another content of section 1.</p> -->
                        <hr>
                        <h2>Section 2</h2>
                        <p>This is the content of section 2.</p>
