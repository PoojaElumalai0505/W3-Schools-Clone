CSS Margins

❮ Previous
Next ❯

CSS Margin property is used to define the space around elements. It is completely transparent and doesn't have any background color. It clears an area around the element.


Margin: Shorthand Property

CSS shorthand property is used to shorten the code. It specifies all the margin properties in one property.

There are four types to specify the margin property. You can use one of them.

  1. margin: 50px 100px 150px 200px;

    2. top margin value is 50px, right margin value is 100px, bottom margin value is 150px, left margin value is 200px

  2. margin: 50px 100px 150px;

    3. top margin value is 50px, left and right margin values are 100px, bottom margin value is 150px

  3. margin: 50px 100px;

    4. top and bottom margin values are 50px, left and right margin values are 100px

  4. margin 50px;

    5. top right bottom and left margin values are 50px

                            
Example

                            <style> 
                                selector{
                                    margin: 50px 100px 150px 200px;  
                                }
                            </style>
