Javascript Comments

❮ Previous
Next ❯

JavaScript comments are non-executable lines of text that you can include in your code to provide explanations, documentation, or notes.

Comments are ignored by the JavaScript interpreter and are not executed as part of the program.

They are helpful for developers to understand the code's logic, purpose, and any relevant information.


Advantages of Javascript Comments:

  • To make code easy to understand.
  • To avoid the unnecessary code.

Types of JavaScript Comments:

  1. Single-line Comment
  2. Multi-line Comment

Single-line Comment:

                
Example

                <script>
                    /* This is a single line comment */
                </script>

Multi-line Comment:

                
Example

                <script>
                    /* It is multi line comment.  
                    It will not be displayed */
                </script>
