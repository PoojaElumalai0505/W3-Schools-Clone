JavaScript is a programming language used for creating interactive and dynamic content on websites.

            
Example

            <script> 
              document.write("Hello JavaScript");  
            </script>

How to execute Javascript?

  • Javascript can be executed right inside one's browser. You can open the Javascript console and writing Javascript there
  • Another way to execute Javascript is a runtime like Node.js which can be installed and used to run Javascript code.
  • Another way to execute javascript is by inserting it inside <script> tag of an HTML document.

Features:

  • It is a case-sensitive language.
  • JavaScript is supportable in several operating systems including, Windows, macOS, etc.
  • It provides good control to the users over the web browsers.
  • It is a light-weighted and interpreted language.

    • Applications:

  • Client-side validation.
  • Dynamic drop-down menus.
  • Displaying date and time.
  • Displaying pop-up windows and dialog boxes (like an alert dialog box, confirm dialog box and prompt dialog box).
